# Quickstart: World-Class Premium Lesson UI

## What This Builds

A complete upgrade of the ScholarX lesson page to Apple-caliber quality across 7 dimensions:
1. Token-based design system
2. Centralized motion library
3. Smart progress tracking with heatmap
4. Focus Mode
5. Expanded glassmorphic primitives
6. Spatial 3-layer layout with overlays
7. Performance-hardened interaction model

## Prerequisites

- Next.js dev server running (`doppler run -- pnpm dev`)
- Navigate to any lesson: `http://localhost:3000/courses/[slug]/lessons/[lessonId]`
- Existing packages already installed: `framer-motion`, `zustand`, `vaul`, `@vidstack/react`

## No New Dependencies Required

All 7 phases use existing packages. Zero new `pnpm add` calls.

## Implementation Order

Follow `tasks.md` (generated by `/speckit-tasks`). Critical dependency chain:

```
Phase 1 (Foundation) must complete before Phase 2+
  └─ design-tokens.ts     → consumed by all UI components
  └─ motion-variants.ts   → consumed by all animated components
  └─ ui-layout-store.ts   → consumed by overlay components + focus mode

Phase 2 (Primitives) must complete before Phase 3+
  └─ glass-panel.tsx      → used by sidebar, notes, resources
  └─ animated-button.tsx  → used by all interactive controls

Phase 3 (Layout) can be done in parallel with Phase 4
Phase 4 (New Overlays) can be done in parallel with Phase 3
Phase 5 (Motion Audit) depends on Phase 3 + 4
Phase 6 (Performance) wraps up after Phase 5
Phase 7 (Accessibility) cross-cuts all phases — apply alongside each
```
